# SpecPilot MVP Implementation Plan

> For agentic workers: implement this plan milestone by milestone. Do not skip ahead. Do not implement features listed as deferred or P2 unless the user explicitly requests them.

**Goal:** Build the SpecPilot MVP as a local full-stack system that generates zero-locator test scenarios from the 4ga Boards manual, executes them with browser-use, verifies results, and visualizes the whole workflow.

**Architecture:** Next.js provides the Chinese testing console. FastAPI owns ingestion, scenario generation, run orchestration, SSE, verification, artifacts, and reports. LangGraph wraps the post-scenario workflow while browser-use remains the only browser execution loop.

**Tech Stack:** Next.js App Router, TypeScript, Tailwind CSS, Radix UI, Lucide Icons, React Flow, TanStack Query, Zustand, Recharts, FastAPI, SQLModel, SQLite, ChromaDB, LangGraph, browser-use, DeepSeek V4 Pro by default, optional Browser Use hosted LLM provider, GLM-4.6V.

---

Implement:

- scenario validator that recursively rejects forbidden locator fields;
- ID helpers for `ft_`, `sc_`, `run_`, `crawl_`, `idx_`, `job_`.
- text LLM provider settings: `TEXT_LLM_PROVIDER=openai_compatible` by default, `OPENAI_COMPATIBLE_BASE_URL`, `OPENAI_COMPATIBLE_API_KEY`, and `OPENAI_COMPATIBLE_MODEL`; legacy `deepseek` and optional `browser_use` provider settings remain readable for compatibility.
- `BROWSER_USE_CLOUD_BROWSER_ENABLED` must default to `false` and must not be used by MVP run execution.

Test requirements:

- valid scenario passes;
- any nested `selector`, `locator`, `xpath`, `element_id`, `element_index`, `css_selector` field fails;
- `.env.example` keys map to settings fields;
- default confidence thresholds are `0.85` and `0.60`.
- default text provider is OpenAI-compatible, default OpenAI-compatible model is `gpt-5.5`, and default DeepSeek compatibility model is `deepseek-v4-pro`;
- Browser Use hosted LLM key is optional unless `TEXT_LLM_PROVIDER=browser_use` or fallback is enabled.

Acceptance:

```bash

uv run ruff check .
```
